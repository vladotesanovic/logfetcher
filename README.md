# LogFetcher

Simple program for querying logs on Ubuntu system.

## Installation

```bash
git clone https://github.com/vladotesanovic/logfetcher.git

cd logfetcher

# Install all dependencies
go get

# Build 
go build
go install

```

## Use

```bash
./logfetcher "Search term"

```
